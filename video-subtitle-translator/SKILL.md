# 多语言字幕翻译器

## 简介
专业的视频字幕提取与多语言翻译工具，支持从视频中提取字幕、翻译成多国语言、生成双语字幕文件。适合做跨境内容、本地化运营、国际搬运的创作者。

- "生成双语字幕"
- "字幕提取"
- "翻译video字幕"

## 核心功能

### 1. 字幕提取
- 支持格式：SRT、ASS、VTT、TXT
- 优先提取视频内嵌字幕
- 自动识别字幕语言
- 支持时间轴精确提取

### 2. 多语言翻译
支持翻译语言：
- 中文（简体/繁体）
- 英语
- 日语
- 韩语
- 法语
- 德语
- 西班牙语
- 葡萄牙语
- 俄语
- 阿拉伯语
- 越南语
- 泰语
- 更多语言...

### 3. 输出格式

#### 3.1 纯翻译字幕
仅目标语言字幕，适合配音/翻译腔视频

#### 3.2 双语字幕
原语言+目标语言上下/左右排列
```
[00:00:01] 你好，欢迎观看
Hello, welcome to watch
```

#### 3.3 三语字幕
适合多语言需求场景

### 4. 字幕样式选择
- 经典白字黑边
- 动态字幕
- 高亮重点词
- 卡拉OK效果

## 使用场景

### 场景1：国际搬运
```
用户：把YouTube英文视频翻译成中文
→ 提取英文字幕
→ 翻译成中文
→ 生成中文SRT文件
```

### 场景2：跨境电商
```
用户：产品视频需要日语字幕
→ 提取原字幕或用户提供文案
→ 翻译成日语
→ 生成日语字幕文件
```

### 场景3：外语学习
```
用户：生成双语字幕学习视频
→ 提取原字幕
→ 翻译成目标语言
→ 生成双语对照字幕
```

### 场景4：内容本地化
```
用户：我的视频需要翻译成5个语言
→ 自动生成5个语言的字幕文件
→ 支持批量处理
```

## 技术说明
- 字幕提取依赖视频自带字幕轨道
- 无字幕视频需先使用语音识别生成字幕
- 翻译质量取决于AI翻译模型能力
- 建议对重要内容人工校对
