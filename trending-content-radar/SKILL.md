# 热点选题雷达

## 技能定位
专为短视频创作者设计的多平台热点追踪工具，聚合抖音、快手、B站、小红书、视频号等平台热点数据，智能分析并推荐创作选题。

## 核心功能

### 1. 多平台热点聚合
实时追踪以下平台热点：
- **抖音**：热搜榜、挑战榜、飙升榜
- **快手**：热搜榜、创意榜
- **B站**：热门视频、飙升视频
- **小红书**：热门话题、热搜词
- **视频号**：朋友圈热点、话题榜
- **微博**：热搜榜、娱乐榜
- **知乎**：热榜、潜力问题

### 2. 选题推荐引擎
基于热点生成创作选题：
- **话题延伸**：将大热点拆解为小角度
- **领域适配**：根据创作者领域过滤热点
- **时效性排序**：判断热点生命周期
- **竞争度分析**：预估选题竞争程度

### 3. 爆款内容预测
分析热点背后的内容规律：
- 内容类型（教程/娱乐/情感/干货）
- 时长适配（15秒/60秒/3分钟）
- 切入角度推荐
- 标题方向建议

### 4. 选题价值评估
多维度评估选题潜力：
- 热度指数（0-100）
- 创作难度（1-5星）
- 变现潜力（1-5星）
- 生命周期（短/中/长）

## 使用方式

### 输入
用户提供：
- 创作领域（可选）
- 目标平台（可选）
- 需要推荐的选题数量
