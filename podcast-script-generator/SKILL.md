# 播客脚本生成器 (Podcast Script Generator)

## 产品定位
帮助播客主理人、内容创作者快速生成专业播客脚本的工具。支持单人和双人对话模式，生成结构完整、节奏把控到位的播客内容。

## 核心功能
- 根据主题生成完整播客脚本
- 支持单人/双人对话模式
- 生成开场白、核心内容、结尾互动
- 提供时长估算和节奏建议
- 自动添加转场和停顿提示

## 使用场景
1. 知识分享类播客（教程、科普、干货）
2. 访谈类播客（嘉宾对话）
3. 故事叙述类播客（个人经历、行业故事）
4. 新闻评论类播客（热点解读）

## 输出格式
```
【基本信息】
- 标题：[播客标题]
- 模式：[单人/双人]
- 预估时长：[X]分钟
- 主播：[主播名]

【开场白】
[吸引听众的开场内容，约15-30秒]

【本期主题】
[一句话概括本期内容]

【核心内容】
## 章节1: [章节标题] (~X分钟)
[具体内容...]
- 关键观点1
- 关键观点2
- 案例/故事

## 章节2: [章节标题] (~X分钟)
...

【互动环节】
[引导听众互动的内容]

【结尾】
[总结+感谢+下期预告]
```

## 技能优势
- 结构清晰，逻辑完整
- 节奏把控专业
- 开场和结尾模板化
- 双人对话自然流畅
