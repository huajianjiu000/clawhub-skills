# 短视频数据指标说明

## 核心指标

### 1. 播放量（VV/UV）
- **定义**：视频被播放的次数（VV）或人数（UV）
- **重要性**：基础流量指标，决定初始推荐量
- **参考值**：
  - 新账号：500-2000
  - 成长期：1万-10万
  - 成熟期：10万-100万
  - 爆款：100万+

### 2. 点赞数
- **定义**：用户点击点赞的次数
- **重要性**：反映内容认可度，影响推荐
- **参考比率**：点赞/播放 = 3%-10%
  - 低于3%：内容需要优化
  - 3%-8%：正常水平
  - 8%-15%：优质内容
  - 高于15%：爆款潜力

### 3. 评论数
- **定义**：用户在视频下方留言的数量
- **重要性**：互动指标，反映内容讨论度
- **参考比率**：评论/播放 = 0.5%-3%
  - 低于0.5%：互动性弱
  - 0.5%-2%：正常
  - 高于2%：高互动

### 4. 转发/分享数
- **定义**：用户将视频分享到其他平台的次数
- **重要性**：社交裂变指标，高转发=强传播
- **参考比率**：转发/播放 = 0.5%-2%

### 5. 收藏/保存数
- **定义**：用户收藏或保存视频的数量
- **重要性**：用户价值认可，实用内容高收藏
- **参考比率**：收藏/播放 = 1%-5%

### 6. 关注转化
- **定义**：用户看完视频后关注账号的数量
- **重要性**：账号增长指标
- **参考比率**：关注/播放 = 0.5%-2%

## 互动指标

### 1. 互动率
```
互动率 = (点赞 + 评论 + 转发 + 收藏) / 播放量 × 100%
```
- **优秀**：> 10%
- **良好**：5%-10%
- **一般**：3%-5%
- **较差**：< 3%

### 2. 完播率
```
完播率 = 完整观看次数 / 播放量 × 100%
```
- **抖音**：
  - 30秒内视频：> 50% 优秀
  - 1分钟视频：> 30% 优秀
  - 3分钟视频：> 15% 优秀
- **快手**：完播率相对较低，参考值约为抖音的60%
- **视频号**：参考值约为抖音的80%

### 3. 复播率
- **定义**：同一用户多次观看同一视频
- **重要性**：反映内容吸引力和用户粘性
- **优秀标准**：> 1.5次/人

### 4. 5秒完播率
- **定义**：前5秒留存率
- **重要性**：封面和开头质量指标
- **优秀标准**：> 60%

## 账号指标

### 1. 粉丝量
- 账号规模划分：
  - 素人：< 1万
  - KOC：1万-10万
  - KOL：10万-100万
  - 头部：100万+

### 2. 粉丝活跃度
```
活跃度 = 月互动粉丝 / 总粉丝 × 100%
```
- **优秀**：> 5%
- **良好**：2%-5%
- **一般**：1%-2%

### 3. 粉丝增长趋势
- 日均增长：> 100 优秀
- 周均增长：> 500 优秀
- 月均增长：> 2000 优秀

## 算法相关

### 1. 推荐流量池等级
| 等级 | 播放量 | 说明 |
|------|--------|------|
| 初始流量池 | 200-500 | 冷启动 |
| 二次分发 | 3000-5000 | 初始测试 |
| 千人千面 | 1万-10万 | 中级推荐 |
| 万人推荐 | 10万-100万 | 高级推荐 |
| 爆款池 | 100万+ | 全网推送 |

### 2. 权重因素
- 完播率：高权重
- 互动率：中权重
- 关注转化：中权重
- 分享率：高权重
- 负面反馈：负权重

### 3. 降权原因
- 搬运/抄袭内容
- 诱导关注/点赞
- 敏感话题/违禁词
- 低质量内容
- 过度营销
