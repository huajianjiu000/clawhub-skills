# 数据分析框架指南

## 经典分析模型

### 1. AARRR模型（海盗模型）

#### Acquisition（获取）
- 播放量来源
- 推荐流量 vs 搜索流量
- 站外引流效果

#### Activation（激活）
- 5秒完播率
- 前3秒留存
- 封面点击率

#### Retention（留存）
- 粉丝回访率
- 复播率
- 账号粘性

#### Referral（推荐）
- 转发分享数
- 社交裂变效果
- 内容传播度

#### Revenue（变现）
- 商品点击率
- 小店访问量
- 广告收益

### 2. 漏斗分析模型

```
曝光 → 点击 → 观看 → 互动 → 关注 → 转化
```

每层转化率参考：
- 点击率：3%-10%
- 完播率：30%-50%
- 互动率：3%-8%
- 关注率：1%-3%
- 转化率：0.5%-2%

### 3. 归因分析

#### 单一归因
- 爆款原因归因单一因素

#### 多重归因
- 标题 + 内容 + 时机 + 运气
- 权重分配评估

## 分析维度

### 1. 内容维度
- 视频类型分析
- 时长效果对比
- 系列内容表现
- 热点响应效果

### 2. 时间维度
- 发布时间分析
- 最佳时段测试
- 节假日表现
- 周期性规律

### 3. 受众维度
- 粉丝画像
- 地域分布
- 性别年龄
- 兴趣标签

### 4. 竞品维度
- 同行数据对比
- 爆款特征提取
- 差异化分析

## 分析流程

### 1. 数据收集
```
步骤1：确定分析目标
步骤2：收集原始数据
步骤3：数据清洗整理
步骤4：建立分析指标
```

### 2. 数据分析
```
步骤5：描述性分析（是什么）
步骤6：诊断性分析（为什么）
步骤7：预测性分析（会怎样）
步骤8：规范性分析（怎么做）
```

### 3. 报告输出
```
步骤9：核心发现总结
步骤10：问题诊断
步骤11：优化建议
步骤12：执行计划
```
