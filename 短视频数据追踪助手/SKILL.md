# 短视频数据追踪助手 (Short Video Data Tracker)

## 简介

帮助短视频创作者追踪和分析视频数据，提供播放量、点赞、评论、转化等关键指标的监控和分析报告生成功能。

## 适用场景

- 抖音数据监控
- 快手数据分析
- 视频号数据追踪
- 多平台数据对比
- 爆款视频复盘

## 触发词

```
数据追踪, 播放量分析, 视频数据, 数据报告, 爆款分析, 账号诊断, 数据对比, 趋势分析, KPI追踪, 转化分析
```

## 使用方法

用户提供视频链接或视频基本信息后，自动追踪并分析相关数据。

## 输出格式

### 标准数据分析报告

**基本信息**
- 视频标题：[视频标题]
- 发布平台：[抖音/快手/视频号]
- 发布时间：[日期]
- 视频时长：[X分钟]

**核心数据**
- 播放量：[X万/百万/千万]
- 点赞数：[X]
- 评论数：[X]
- 转发数：[X]
- 完播率：[X%]

**数据解读**
- 数据亮点
- 问题分析
- 改进建议

**对比分析**
- 与账号平均数据对比
- 与同类视频对比
- 趋势判断

**优化建议**
- 标题优化
- 内容优化
- 发布时间优化
- 互动引导

## 核心功能

### 1. 核心指标追踪
- 播放量/浏览量
- 点赞数/喜欢数
- 评论数/弹幕数
- 转发/分享数
- 收藏/保存数
- 关注转化数

### 2. 互动数据分析
- 评论区热词
- 点赞用户画像
- 互动时间段
- 互动率计算

### 3. 完播率分析
- 平均观看时长
- 流失节点分析
- 内容吸引力评估

### 4. 数据报告生成
- 日/周/月报
- 单视频分析
- 账号整体诊断
- 竞品对比分析

### 5. 趋势预测
- 爆款潜力评估
- 推荐算法解读
- 最佳发布时间建议
