# Short Video Hook Generator (短视频开场钩子生成器)

## 1. Concept & Vision

**短视频开场钩子生成器** - 专为抖音、快手、视频号创作者打造的开场3秒钩子文案工具。前3秒决定视频生死，一个好钩子能让观众从滑走变成停留、点赞、关注。

**产品口号**: "3秒留住观众，让流量翻倍"

## 2. Core Features

### 2.1 开场钩子类型库
- **痛点共鸣型**: 直击观众痛点，引发强烈共鸣
- **悬念诱导型**: 制造悬念，让人忍不住看完
- **反常识型**: 违背直觉，引发好奇
- **利益驱动型**: 明确告诉观众能得到什么
- **情绪冲击型**: 强烈情绪开场，调动情感
- **身份认同型**: 精准定位目标人群
- **数据震撼型**: 用数字抓住注意力
- **场景还原型**: 代入感极强的场景开场

### 2.2 适用场景匹配
- 知识干货类
- 搞笑段子类
- 情感励志类
- 美食探店类
- 带货种草类
- 剧情短剧类
- 生活Vlog类

### 2.3 输出格式
每个钩子包含：
- 钩子文案（3-5秒口播内容）
- 适用平台（抖音/快手/视频号）
- 情绪标签（震惊/好奇/温暖/搞笑等）
- 使用建议（配合什么画面效果更佳）

## 3. User Interaction Flow

```
用户输入 → 选择内容类型 → 生成10+钩子选项 → 用户选择/修改 → 输出完整开场文案
```

**用户输入示例**:
- "职场穿搭" → 生成职场女性穿搭技巧的开场钩子
- "省钱技巧" → 生成省钱妙招类视频的吸引钩子

**输入**: 美食探店 - 重庆火锅
**输出**:
```markdown
🔥 钩子1: "老板说这家店再不来就倒闭了！"
   类型: 悬念诱导型
```

## 5. Technical Specifications

- **输入**: 内容主题/行业/类型
- **处理**: 基于海量爆款视频数据分析的钩子模板库
- **输出**: 结构化钩子文案 + 使用建议
- **语言**: 中文为主，支持中英双语
