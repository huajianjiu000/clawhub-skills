# 短视频热点追踪助手

## 功能描述

专为短视频创作者打造的全平台热点追踪工具，帮助创作者快速发现抖音、快手、视频号、B站等平台的热门内容趋势，提供选题灵感和内容方向参考。

## 适用场景

- 短视频创作者需要追踪热点寻找选题
- 内容运营人员需要了解平台热门趋势
- 自媒体人需要分析竞品爆款内容
- 品牌方需要监测行业热点话题

## 核心能力

### 支持平台
- **抖音**：热榜话题、热门视频、挑战赛
- **快手**：热榜视频、达人榜单、热门话题
- **视频号**：热门视频、话题榜单
- **B站**：热门视频、话题榜、排行榜
- **全平台**：跨平台热点聚合

### 功能特点
1. **热点聚合**：实时聚合各平台热榜内容
2. **话题分析**：深度分析热点话题的传播路径和内容特征
3. **趋势预测**：基于历史数据预测热点持续时间
4. **选题推荐**：根据热点推荐相关选题方向
5. **竞品分析**：分析同类账号的爆款内容规律

## 使用方式

### 基本用法
```
今日有哪些热点可以追？
帮我分析一下最近的短视频热门趋势
```

### 进阶用法
```
分析[领域]的热点，比如：美食、旅游、穿搭
找出[平台]上[时间范围]内的爆款视频特征
给我推荐几个[类型]的选题方向
```

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| 追踪类型 | 是 | 热点/话题/竞品/趋势 |
| 目标平台 | 否 | 指定平台，默认为全平台 |
| 时间范围 | 否 | 今日/本周/本月 |
| 内容领域 | 否 | 美食/旅游/科技等 |

...

## 技术实现

本技能通过以下方式工作：

1. **数据采集**：聚合各平台公开热榜数据
2. **热度计算**：基于播放量、点赞、评论等指标计算热度
3. **趋势预测**：分析历史数据预测热点生命周期
4. **选题匹配**：根据热点关联度推荐选题方向
