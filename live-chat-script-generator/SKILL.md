# Live Chat Script Generator (直播互动话术生成器)

## 1. Concept & Vision

**直播互动话术生成器** - 帮助主播打造高互动、高转化直播的话术工具。从欢迎话术、弹幕互动、促单逼单到下播感谢，全方位提升直播间氛围和转化率。

**产品口号**: "让每一场直播都成为爆款"

## 2. Core Features

### 2.1 话术类型库

#### 欢迎互动类
- 新人进场欢迎话术
- 老粉回访暖场话术
- 粉丝团等级差异化欢迎

#### 弹幕互动类
- 提问回答话术
- 弹幕节奏把控
- 互动福利引导

#### 促单转化类
- 产品介绍话术（FAB法则）
- 限时限量逼单话术
- 对比算账促单话术
- 解决顾虑话术

#### 气氛营造类
- 憋单话术
- 放单话术
- 抽奖互动话术

#### 下播收尾类
- 感谢话术
- 预告下期话术
- 引导关注话术

### 2.2 适用场景
- 带货直播（美妆/服装/食品/数码）
- 知识付费直播
- 娱乐秀场直播
- 品牌专场直播

### 2.3 输出格式
```
【场景】xxx
【话术】完整的直播话术文本
【技巧】使用注意要点
【配合】建议配合的互动动作
```

## 3. User Interaction Flow

```
选择直播场景 → 选择话术类型 → 生成完整话术 → 根据实际情况调整使用
```

**用户输入示例**:
- "美妆带货直播" + "促单话术" → 输出完整的美妆产品促单话术

## 4. Examples

**输入**: 服装直播 - 催单逼单
**输出**:
```markdown
【场景】服装直播 - 限时催单

【话术】
"好了好了，倒计时开始！3！2！1！
这波福利只给你们5分钟！
别跟我说再想想，等你想好了库存就没了！
小黄车1号链接，手慢无！
倒计时结束直接关链接，我说到做到！

【技巧】
1. 倒计时要配合紧迫BGM
2. 说完"关链接"要真的准备关
3. 主播眼神要坚定，不要犹豫

【配合动作】
- 手指向小黄车
- 假装要点击关闭按钮
- 身体微微前倾，语气加速
```

## 5. Technical Specifications

- **输入**: 直播类型 + 产品类别 + 话术场景
- **处理**: 基于头部直播间话术库分析提炼
- **输出**: 结构化完整话术 + 使用技巧
- **语言**: 中文
