# 智能会议纪要生成器

## 功能描述

智能会议纪要生成工具，能够自动分析会议录音或文字记录，提取关键信息，生成结构化会议纪要。支持识别发言人、提取决策点、生成待办事项和时间节点。

## 适用场景

- 企业会议记录需要快速生成纪要
- 团队协作需要明确行动项和责任人
- 项目管理需要追踪会议决策和进度
- 个人需要高效整理会议要点

## 核心能力

### 功能特点
1. **智能识别**：自动识别发言人，区分不同角色的观点
2. **决策提取**：从冗长讨论中提取关键决策和结论
3. **待办生成**：自动识别行动项，指定责任人和截止时间
4. **结构化输出**：生成标准化的会议纪要模板
5. **多格式支持**：支持文本、录音转文字等多种输入

### 输出结构
- 会议基本信息（时间、主题、参会人员）
- 讨论要点汇总
- 关键决策记录
- 行动待办清单
- 下次会议安排

## 使用方式

### 基本用法
```
帮我整理这个会议纪要
根据以下内容生成会议摘要
```

### 进阶用法
```
分析以下会议内容，提取：
1. 关键决策点
2. 待办事项（含责任人和截止时间）
3. 未解决的问题

会议内容：[粘贴会议记录]
```

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| 会议内容 | 是 | 会议记录文字或录音转写文本 |
| 会议主题 | 否 | 自动识别或手动指定 |
| 参会人员 | 否 | 自动识别或手动指定 |
| 输出格式 | 否 | 详细/简洁/要点 |

## 技术实现

本技能通过以下方式工作：

1. **内容解析**：分析会议文本，识别语义和结构
2. **角色识别**：根据上下文识别不同发言人
3. **关键信息提取**：提取决策、待办、问题等关键信息
4. **结构化生成**：按标准模板生成结构化纪要
5. **待办追踪**：生成可追踪的行动清单
