# AI文案去味器 V2 (AI Copy Deodorizer V2)

## 技能概述
专门解决AI生成文本"一眼假"的问题，自动检测并修复AI写作中常见的模式，让AI生成的内容更自然、更有人情味。

## 核心功能

### 1. AI痕迹检测
自动识别以下AI写作模式：
- 夸大象征意义（"正如...所证明的"）
- 宣传性语言（"令人惊叹的"、"突破性的"）
- 肤浅分析（"这说明我们需要..."）
- 破折号过度使用
- 三段式法则（观点-解释-总结）
- AI词汇（"首先"、"其次"、"最重要的是"）
- 否定式排比（"不是...而是..."）
- 过度礼貌用语
- 机械的过渡词

### 2. 智能改写
将检测到的AI模式转换为更自然的表达：
- 使用更口语化的表达
- 增加个人视角和情感
- 减少过度结构化
- 加入真实案例和细节
- 使用更自然的过渡

### 3. 风格适配
根据目标场景调整文案风格：
- 社交媒体：活泼、简短、有互动感
- 正式场合：专业、严谨、但不死板
- 私域运营：亲切、真诚、有温度
- 短视频口播：口语化、节奏感强

## 适用场景
- 小红书/抖音文案优化
- 朋友圈推广文案
- 私聊话术改写
- 视频脚本润色

## 使用方法
用户输入AI生成的文案 → 输出改写后的自然文案+修改说明

## 输出格式
```
🤖 AI痕迹检测
检测到以下AI模式：
- [模式1]
- [模式2]
...

✏️ 改写建议
原句：[原文]
→ 改写：[新文]

📝 改写后完整文案
[完整改写后的内容]

💡 修改要点
1. [要点1]
2. [要点2]
...
```
