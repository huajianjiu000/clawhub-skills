# 封面配色方案指南

## 高点击率配色原则

1. **高对比度**：文字与背景形成强烈对比
2. **色彩饱和度**：适当提高饱和度，更吸引眼球
3. **品牌一致性**：保持账号整体风格统一
4. **节假日适配**：节日主题配色提升时效性

## 配色方案推荐

### 方案1：红色+白色（高点击）
```
背景色：#FF4757（红色）
文字色：#FFFFFF（白色）
适用：促销/限时/热点
```

### 方案2：深色+亮色（高级感）
```
背景色：#2C3A47（深灰）
文字色：#FFD93D（亮黄）
适用：知识/干货/教程
```

### 方案3：渐变色（现代感）
```
背景：线性渐变 #667EEA → #764BA2
文字色：#FFFFFF（白色）
适用：科技/时尚/创意
```

### 方案4：黄色+黑色（强对比）
```
背景色：#FBC531（黄色）
文字色：#000000（黑色）
适用：搞笑/猎奇/反转
```

## 配色注意事项

1. **避免文字看不清**：白色文字配深色背景，黑色文字配浅色背景
2. **不要超过3种颜色**：主色+辅色+强调色
3. **考虑色盲用户**：红绿搭配要谨慎
4. **保持品牌调性**：与账号整体风格一致
