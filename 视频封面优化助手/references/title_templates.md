# 封面标题模板库

## 爆款标题类型

### 1. 数字型标题
```
格式：数字 + 痛点/利益点
示例：
- "3个技巧，让你的视频播放量翻倍"
- "99%的人都不知道的5个隐藏功能"
- "月薪3000和30000的人差在哪里？"
```

### 2. 悬念型标题
```
格式：制造好奇 + 引导点击
示例：
- "最后一步90%的人都会忽略..."
- "这个视频可能会改变你的人生"
- "看完这个视频，我失眠了3天"
```

### 3. 痛点型标题
```
格式：戳中痛点 + 提供方案
示例：
- "还在为XX烦恼？看完你就明白了"
- "为什么你做的XX总是不成功？"
- "XX的3个致命错误，你中了几个？"
```

### 4. 利益型标题
```
格式：明确利益 + 快速获取
示例：
- "免费送你一份XX攻略，点击领取"
- "学会这招，XX问题轻松解决"
- "XX天学会XX，全套教程拿走"
```

### 5. 情感型标题
```
格式：引发共鸣 + 情绪代入
示例：
- "这就是我不想生孩子的理由"
- "成年人的崩溃，往往就在一瞬间"
- "感谢曾经的看不起，如今让你高攀不起"
```

### 6. 反转型标题
```
格式：反转预期 + 出乎意料
示例：
- "月薪3000的工作，我为什么抢着做？"
- "吃遍全城的火锅店，最后被这家路边摊征服"
- "985毕业去送外卖，家人骂我没出息..."
```

## 各平台标题风格

### 抖音标题特点
- 口语化、接地气
- 短句为主，便于阅读
- 情绪化、夸张化
- 常用"竟然"、"原来"、"竟然"等词

### 快手标题特点
- 更接地气
- 方言可用
- 直接表达
- 强调真实

### B站标题特点
- 可以稍微长一点
- 知识型内容多
- 二次元梗可用
- "万恶之源"、"前方高能"等

### 视频号标题特点
- 偏朋友圈风格
- 情感类内容多
- 转发分享导向
- 偏中老年审美

## 标题字数建议

| 平台 | 主标题 | 副标题 | 总字数 |
|------|--------|--------|--------|
| 抖音 | 8-15字 | 15-20字 | 20-30字 |
| 快手 | 10-18字 | 15-25字 | 25-35字 |
| B站 | 15-25字 | 20-30字 | 35-50字 |
| 视频号 | 12-20字 | 15-25字 | 25-40字 |

## 标题公式

### 通用公式
```
痛点/利益 + 数字 + 悬念 + 情绪
```

### 实用公式
```
1. 【数字】+ 【结果】：X个技巧，让你的XX提升XX倍
2. 【悬念】+ 【揭秘】：XX的真相竟然是...（揭秘）
3. 【痛点】+ 【解决】：还在为XX烦恼？3步教你解决
4. 【对比】+ 【反转】：XX和XX的差距，颠覆你的认知
```

## 标题避坑指南

1. **不要标题党**：承诺的内容视频里要有
2. **不要夸大**：避免"最全"、"第一"等极限词
3. **不要低俗**：违规词会被限流
4. **不要太长**：移动端显示有限，过长会被截断
5. **不要重复**：多准备几个标题版本测试
