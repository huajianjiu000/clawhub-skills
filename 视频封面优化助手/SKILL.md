# 视频封面优化助手 (Video Cover Optimizer)

## 简介

帮助短视频创作者优化视频封面，提升点击率和观看量。提供封面设计建议、标题优化、配色方案等专业指导。

## 适用场景

- 抖音/快手/视频号封面设计
- B站视频封面优化
- 视频号封面标题设计
- 自媒体封面制作

## 触发词

```
封面优化, 视频封面, 封面设计, 标题优化, 点击率提升, 封面配色, 缩略图优化, 封面文案, 视频预览图
```

## 使用方法

用户提供视频主题、目标受众、平台信息后，生成封面优化方案。

## 输出格式

### 标准封面优化方案

**基本信息**
- 视频主题：[用户输入]
- 目标平台：[抖音/快手/视频号/B站]
- 目标受众：[年龄/性别/兴趣]
- 当前封面：[用户提供或描述]

**封面诊断**
- 优点分析
- 问题指出
- 优化建议

**优化方案**
- 主标题建议
- 副标题建议
- 配色方案
- 布局建议
- 元素清单

**平台适配建议**
- 各平台封面尺寸
- 最佳发布时间
- 标签建议

## 核心功能

### 1. 封面诊断
- 文字可读性分析
- 色彩对比度检查
- 构图平衡度评估
- 品牌一致性检查

### 2. 标题优化
- 主标题生成（15字内）
- 副标题生成（20字内）
- 悬念式标题
- 数字型标题
- 情感型标题

### 3. 配色方案
- 高点击率配色推荐
- 平台风格适配
- 品牌色系建议
- 节假日主题配色

### 4. 布局建议
- 黄金分割构图
- 文字位置建议
- 人脸/产品展示区
- CTA元素位置
