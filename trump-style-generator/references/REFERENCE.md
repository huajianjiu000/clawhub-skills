# Trump Style Generator - Complete Reference

## Trump's Rhetorical Arsenal

### 1. Power Vocabulary

| Weak Word | Trump Power Word |
|-----------|------------------|
| big | HUGE, TREMENDOUS |
| good | GREAT, THE BEST |
| bad | TOTAL DISASTER |
| many | MANY, MANY, THOUSANDS |
| very | REALLY, ABSOLUTELY |

### 2. Authority Builders
- "Believe me"
- "Trust me on this"
- "Many people are saying"
- "Everyone knows"

### 3. Attack Patterns
- "The failing [X]"
- "Crooked [X]"
- "Sad!"
- "Total disaster"

### 4. Signature Structures

#### The Trump Triad
```
[Claim]. [Repetition]. [Closer].
Example: "It's going to be great. Really great. And we're going to win."
```

#### The Vision Builder
```
"We're going to [action]. It's going to be [adjective]. [Reinforcement]."
```

### 5. Tone Calibration

- **Subtle (10-20%)**: Light power words, one "believe me"
- **Medium (50%)**: Repetition, contrasts, multiple patterns
- **Full Trump (100%)**: Complete arsenal, maximum impact
