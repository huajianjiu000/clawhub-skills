# 短视频话题标签生成器

## 基本信息
- **名称**：短视频话题标签生成器
- **slug**: short-video-hashtag-generator
- **描述**：根据视频主题/内容，智能推荐抖音、快手、视频号等平台的爆款话题标签，提升视频曝光率
- **版本**：1.0.0
- **作者**：沐晴
- **标签**：短视频、运营、标签、抖音、快手

## 功能说明

### 核心能力
根据用户输入的视频主题、类型或内容描述，自动生成：
- **主推标签**（3-5个）：高热度、高相关性标签
- **长尾标签**（5-10个）：精准流量标签
- **蹭热点标签**（1-3个）：当前热门话题标签

### 使用场景
- 短视频发布前：获取最佳标签组合
- 内容策划时：了解热门话题方向
- 账号运营：建立标签库

## 触发方式

### 关键词触发
用户输入以下内容时激活：
- "生成标签"、"推荐标签"、"话题标签"
- "给我标签"、"标签推荐"
- "抖音标签"、"快手标签"

### 询问格式
```
[视频主题/内容描述] + 标签需求
示例：
- "美食探店视频用什么标签好"
- "搞笑段子推荐标签"
- "萌宠日常标签"
```

## 技能配置

### 输出格式
```markdown
📊 **话题标签推荐**

**主推标签**（高热度）
#标签1 #标签2 #标签3

**精准标签**（长尾流量）
#标签4 #标签5 #标签6 #标签7 #标签8

**蹭热点标签**（当前热门）
#标签9 #标签10

💡 使用建议：选择5-8个标签效果最佳
```

### 标签库来源
- 抖音热榜标签
- 快手热门话题
- 视频号趋势标签
- 行业垂直标签

## 使用限制
- 每次最多生成15个标签
- 标签需与内容高度相关
- 建议选择5-8个标签组合使用

## 注意事项
- 标签要与视频内容匹配，避免标签作弊
- 热门标签竞争大，精准标签转化高
- 定期更新标签库，跟上热点变化
