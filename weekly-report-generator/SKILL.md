# 职场周报生成器

## 简介

告别周报写作恐惧症！输入简单的工作内容，自动生成结构清晰、数据突出、领导爱看的专业周报。

## 核心功能

- **智能内容提炼**：从简单描述中提取关键成果
- **数据可视化**：自动将工作内容转化为数据呈现
- **结构化输出**：按照标准周报格式整理
- **多风格适配**：支持国企风、互联网风、创业公司风

## 周报结构

1. **本周完成**：列出主要工作成果
2. **数据亮点**：关键指标的完成情况
3. **问题与挑战**：遇到的困难及解决思路
4. **下周计划**：清晰的下一步工作安排

## 适用人群

- 职场新人（不知道周报怎么写）
- 忙碌的职场人（没时间写周报）
- 想要提升周报质量的任何人
