# 公众号爆款标题生成器

## 功能概述
专为公众号创作者设计的爆款标题生成工具，基于10万+爆款文章标题库训练，生成高打开率的公众号标题。支持多种类型：情感故事、干货分享、热点追评、悬念引导等。

## 核心能力

### 1. 标题类型识别
根据文章主题自动识别最适合的标题类型：
- 情感共鸣型：引发读者情感共振
- 干货价值型：突出实用性和价值感
- 热点追评型：结合时事热点
- 悬念引导型：引发好奇心和点击欲
- 数字列表型：结构清晰，预期明确

### 2. 爆款标题要素
生成标题时自动包含以下要素：
- 情绪词：焦虑、共鸣、好奇、恐惧等
- 数字：具体数据增强可信度
- 关键词：精准触达目标读者
- 悬念点：留白引发点击

### 3. 多版本生成
每次生成5-8个备选标题，包含：
- 主标题（吸引点击）
- 副标题（补充说明）
- 标签建议（便于推荐）

## 输出报告模板

```
【公众号爆款标题生成】
文章主题：{用户输入的主题}

📌 推荐标题 TOP5

1️⃣ {标题1}
   类型：情感共鸣型
   亮点：使用情绪词"xxx"引发共鸣

2️⃣ {标题2}
   类型：数字列表型  
   亮点：具体数字"3步"增强预期感

3️⃣ {标题3}
   类型：悬念引导型
   亮点：留白"真相是..."激发好奇心

4️⃣ {标题4}
   类型：热点追评型
   亮点：结合当下热点"xxx"

5️⃣ {标题5}
   类型：干货价值型
   亮点：突出"保姆级教程"价值感

💡 标题优化建议
- 首句加"曝光！""刚刚"等时效词
- 结尾加"！"或"👇"引导下滑
- 避开平台限流词
```

## 使用场景
1. 公众号文章发布前标题优化
2. 爆款选题的方向确定
3. 多平台内容适配（头条、百家号）
4. 标题A/B测试选优
