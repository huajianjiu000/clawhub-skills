# 每日毒舌语录生成器

## 基本信息
- **名称**：每日毒舌语录生成器
- **slug**: daily-roast-quotes
- **描述**：每日更新毒舌语录/毒鸡汤文案，治愈你的精神内耗。适合发朋友圈、做签名、聊天互动，文案辛辣有趣，句句扎心又清醒
- **版本**：1.0.0
- **作者**：沐晴
- **标签**：毒舌、语录、鸡汤、文案、朋友圈、签名

## 功能说明

### 核心能力
生成各类毒舌文案：
- **毒鸡汤**：反鸡汤、扎心真相
- **毒舌语录**：犀利点评、怼人金句
- **清醒语录**：人间清醒、看透本质
- **摆烂文学**：反焦虑、适度躺平

### 分类标签
- 💀 毒鸡汤
- 🔥 毒舌怼人
- 🧠 人间清醒
- 🛋️ 摆烂文学
- 💼 职场毒舌
- ❤️ 情感毒舌

### 使用场景
- 朋友圈发圈素材
- 社交媒体签名
- 聊天斗图/怼人
- 短视频文案素材
- 头像签名

## 触发方式

### 关键词触发
用户输入以下内容时激活：
- "毒舌语录"、"毒鸡汤"
- "每日语录"、"扎心文案"
- "人间清醒"、"怼人金句"
- "摆烂文案"

### 询问格式
```
[分类] + 数量 + 需求
示例：
- "来几条毒鸡汤"
- "给我5条人间清醒语录"
- "职场毒舌语录"
- "随机来一条扎心的"
```

## 技能配置

### 输出格式
```markdown
🔥 **每日毒舌语录**

💀 【毒鸡汤】
1. 努力不一定会成功，但躺着真的很舒服。
2. 别假装努力，结果不会陪你演戏。
...

🔥 【毒舌怼人】
1. 你那么努力，难怪还是这么穷。
2. 你的智商余额已不足，请及时充值。
...

🧠 【人间清醒】
1. 不要总想着改变别人，先改变自己。
2. 成年人最好的默契是不言而喻。
...

💡 今日金句：「你之所以焦虑，是因为懒」
```

### 语录库规模
- 毒鸡汤语录：100+条
- 毒舌怼人：80+条
- 人间清醒：100+条
- 摆烂文学：50+条
- 职场/情感毒舌：各50+条

## 使用限制
- 每次最多生成20条
- 支持按分类筛选
- 可指定风格比例

## 注意事项
- 毒舌文案仅供娱乐，请勿当真
- 怼人使用注意场合和对象
- 部分语录可能过于犀利，请酌情使用
