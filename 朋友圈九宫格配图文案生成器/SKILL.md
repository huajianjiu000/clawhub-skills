# 朋友圈九宫格配图文案生成器

## 基本信息
- **名称**：朋友圈九宫格配图文案生成器
- **slug**: moments-grid-copywriter
- **描述**：为朋友圈九宫格图片生成配套文案，支持多种风格：文艺、清新、搞笑、情感、商务，让你的朋友圈更吸睛
- **版本**：1.0.0
- **作者**：沐晴
- **标签**：朋友圈、文案、社交、运营

## 功能说明

### 核心能力
根据用户选择的场景和风格，生成：
- **九宫格统一文案**：适合9张图的统一主题文案
- **每图配文**：为每张图提供单独配文
- **统一话题标签**：朋友圈话题标签建议

### 支持风格
1. **文艺清新**：简约、有意境、文艺气息
2. **搞笑幽默**：段子手风格、逗趣吸睛
3. **情感治愈**：温暖、打动人心
4. **商务专业**：职场、正式、干练
5. **生活记录**：日常、接地气
6. **旅行打卡**：风景、地点、攻略

### 使用场景
- 旅行打卡发圈
- 美食分享
- 日常记录
- 产品推广
- 个人IP打造

## 触发方式

### 关键词触发
用户输入以下内容时激活：
- "朋友圈文案"、"发圈文案"
- "九宫格文案"、"配图文案"
- "朋友圈素材"

### 询问格式
```
[场景/内容] + [风格偏好] + 文案需求
示例：
- "周末下午茶九宫格文案"
- "旅行打卡文艺风格文案"
- "美食九张图配文"
```

## 技能配置

### 输出格式
```markdown
🎨 **朋友圈九宫格配图文案**

**风格**：文艺清新
**主题**：周末下午茶时光

**统一文案**：
"偷得浮生半日闲，一杯茶、一本书、一个下午"

**分图配文**：
1️⃣ 今日份的小确幸
2️⃣ 咖啡香里的悠闲时光
3️⃣ ...
9️⃣ 下次见~

**话题标签**：
#周末碎片 #下午茶时光 #慢生活

💡 发圈小技巧：配图色调统一更美观
```

## 使用限制
- 支持1-9张图的配文生成
- 风格选择可多选组合
- 单次最多生成3种风格对比

## 注意事项
- 文案需符合朋友圈社交礼仪
- 商业推广注意频率和方式
- 图片建议统一色调和风格
