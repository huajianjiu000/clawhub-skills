# 外卖优惠券查询助手

## 功能概述
聚合主流外卖平台（美团、饿了么）实时优惠券信息，智能匹配最优红包，支持红包叠加计算，帮助用户在点外卖时省到最多。

## 核心能力

### 1. 平台优惠券聚合
- 美团外卖：天天神券、会员红包、新客券
- 饿了么：超级吃货卡、店铺红包、新用户专享
- 天天领红包：每日签到红包、限时抢购
- 店铺满减：店铺自有优惠活动

### 2. 智能红包匹配
根据用户输入的信息：
- 输入店铺名称 → 查找该店可用优惠
- 输入预算金额 → 推荐最省方案
- 输入菜品清单 → 计算最优凑单

### 3. 红包叠加计算
自动计算：
- 平台红包 + 店铺满减 + 会员红包
- 找到最佳叠加方案
- 显示预估节省金额

### 4. 实时优惠提醒
- 整点红包雨提醒
- 大额红包首发通知
- 即将过期红包预警

1. 每日点外卖前查询优惠
