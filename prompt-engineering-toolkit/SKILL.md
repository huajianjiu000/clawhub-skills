# AI Prompt Optimizer

> 🚀 Transform your prompts from good to great

## Skill Overview

This skill helps users optimize their AI prompts for better, more consistent results. It analyzes prompt structure, identifies weaknesses, and provides enhanced versions with explanations.

## Core Capabilities

- **Prompt Analysis**: Evaluate clarity, specificity, and structure
- **Multi-model Support**: Optimize for ChatGPT, Claude, Gemini, and more
- **Use Case Templates**: Coding, writing, analysis, creative tasks
- **Iterative Improvement**: Generate multiple optimized versions
- **Best Practices**: Apply prompt engineering principles automatically

## Trigger Keywords

- `/optimize-prompt`
- `/improve-prompt`
- `/prompt-help`
- `/better-prompt`

## Usage Examples

### Example 1: Basic Optimization
**Input**: "Write a blog post about AI"

**Output**: 
```
Optimized Prompt:
Write a 1000-word blog post about the impact of AI on small businesses. 
Include:
- 3 real-world examples
- Statistics from 2023-2024
- A section on future predictions
- Practical tips for business owners

Target audience: Small business owners with basic tech knowledge.
Tone: Informative but accessible.
```

### Example 2: Coding Prompt
**Input**: "Fix my Python code"

**Output**:
```
Optimized Prompt:
Please analyze and fix the following Python code:
[PASTE CODE]

Provide:
1. List of bugs/errors found
2. Fixed code with comments explaining changes
3. Best practices recommendations
4. Test cases to verify the fix
```

## How to Use

1. Share your original prompt
2. Specify the AI model you're using (optional)
3. Describe your desired outcome
4. Receive optimized prompts with explanations

```markdown
## Prompt Analysis

- Structure Score: X/10

## Issues Found

## Optimized Prompt
[Enhanced version]

[Explanation of improvements]
```

## Notes

- More specific prompts get better results
- Include context, constraints, and desired format
- Iterative refinement often needed for complex tasks
