# 短视频脚本拆解助手

## 基本信息

- **名称**: video-script-assistant
- **描述**: 专业的短视频文案分析工具，深度拆解爆款视频的脚本结构、钩子设计、情绪节奏和转化策略
- **版本**: 1.0.0
- **适用场景**: 抖音、快手、视频号、TikTok、B站等短视频平台内容分析

## 核心能力

### 1. 脚本结构拆解
将完整脚本分解为标准模块：
- **开场钩子** (0-3秒): 抓注意力
- **痛点/痒点** (3-8秒): 引发共鸣
- **核心内容** (8-45秒): 价值传递
- **高潮反转** (45-55秒): 情绪爆发
- **行动号召** (55-60秒): 引导互动

### 2. 爆款元素分析
- **黄金3秒**: 开场吸引力设计
- **完播率设计**: 节奏把控技巧
- **互动触发**: 评论/点赞引导
- **转化路径**: 关注/购物车/私域引流

### 3. 文案技巧提取
- 话术结构: 疑问句/陈述句/感叹句使用
- 情绪词汇: 高情绪值词汇统计
- 节奏把控: 句式长短搭配
- 语言风格: 口语化/专业感/亲和力

### 4. 模板提炼
从成功案例中提炼可复用的脚本模板

## 输入要求

用户提供以下信息:
1. 视频文案/字幕文本（必需）
2. 视频时长
3. 视频类型（种草/知识/搞笑/剧情等）
4. 目标平台

## 示例

**用户输入**: 
- 文案: "你知道吗？90%的人洗脸都洗错了！难怪皮肤越来越差，用再多精华都没用。今天教你正确洗脸，只需要3步..."
- 时长: 45秒
- 类型: 护肤知识
- 平台: 抖音

**生成结果**:
```
## 📊 短视频脚本拆解报告

### 基础信息
- 视频时长: 45秒
- 视频类型: 护肤知识科普
- 目标平台: 抖音

---

### 🎣 开场钩子分析 (0-3秒)
**原文**: "你知道吗？90%的人洗脸都洗错了！"
**类型**: 数字冲击型 + 痛点型
**吸引力指数**: ⭐⭐⭐⭐⭐
**分析**: 数字"90%"制造冲击感，"洗错了"引发焦虑，激发求知欲

---

### 💡 痛点/痒点设计 (3-8秒)
**原文**: "难怪皮肤越来越差，用再多精华都没用"
**共鸣点**: 
- 戳中护肤焦虑
- 关联用户现状
- 建立"受害者"身份认同

---

### 📌 核心内容结构
| 时间节点 | 脚本内容 | 技巧解析 |
|---------|---------|---------|
| 0-3s | "你知道吗？90%..." | 数字钩子+痛点切入 |
| 3-10s | "难怪皮肤越来越差..." | 强化痛点共鸣 |
| 10-25s | "今天教你正确洗脸..." | 价值承诺 |
| 25-40s | [3步洗脸示范] | 具体方法论 |
| 40-45s | "学会了吗？评论区..." | 互动引导 |

---

### 🎬 节奏把控
- 平均句子长度: 12字（短平快）
- 高潮点位置: 第3秒（痛点爆发）
- 情绪曲线: 焦虑→共鸣→期待→满足→行动

---

### 💬 互动引导设计
**原文**: "学会了吗？评论区告诉我你洗对了吗"
**方式**: 疑问引导型
**效果**: 激发评论欲望，建立互动

---

### 📋 可复用模板
```
[数字冲击] + [痛点强化] + [价值承诺] + [具体方法] + [互动引导]

```

---

### ✅ 优化建议
1. 开头可增加声音特效强化冲击感
2. 3步洗脸可用字幕+动作示范增强记忆
3. 结尾引导关注而非仅评论，增加粉丝转化
```

## 使用限制

- 文本分析基于提供的文案内容
- 视频画面效果无法分析
- 建议结合平台数据验证分析准确性
