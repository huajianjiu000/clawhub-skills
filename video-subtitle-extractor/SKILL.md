# 视频字幕提取工具

## 功能概述
快速从视频链接或本地视频中提取字幕文字，支持多种格式输出。专为短视频创作者设计，方便进行字幕整理、文案存档、内容二次创作。支持SRT、ASS、TXT等多种字幕格式导出。

## 核心能力

### 1. 多源输入支持
- 抖音视频链接
- 快手视频链接
- B站视频链接
- 微信视频号链接
- 本地视频文件（MP4/MKV/AVI）

### 2. 字幕格式支持
- 内嵌字幕（SRT标准格式）
- 弹幕文字提取
- 语音转文字（AI识别）

### 3. 智能处理
- 自动识别字幕语言（中文/英文/中英双语）
- 去除无效时间码和重复内容
- 保留关键语气词和停顿标记
- 自动分段整理

### 4. 输出选项
- 纯文本（TXT）
- 标准字幕（SRT）
- 带时间轴字幕（ASS）
- JSON结构化数据

## 输出报告模板

```
【视频字幕提取工具】

📹 视频来源：{来源平台}
🎬 视频时长：{时长}
🗣️ 识别语言：{语言}

📄 字幕内容预览：
{字幕内容的前200字}

📊 统计信息
- 总字数：{字数}
- 有效字幕行：{行数}
- 预计阅读时间：{时间}

📥 下载选项
1. [TXT文本] - 纯文字，可直接复制使用
2. [SRT字幕] - 标准字幕格式，可用于剪辑软件
3. [ASS字幕] - 带样式字幕，支持特效
4. [JSON] - 结构化数据，便于程序处理

💡 使用建议
- 提取后建议人工校对错别字
- 可配合文案改写工具二次创作
```

## 使用场景
1. 视频文案快速存档
2. 跨平台内容搬运
3. 视频内容改写参考
4. 字幕文件生成
5. 学习笔记整理
