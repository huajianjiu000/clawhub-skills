# 促单逼单技巧指南

## 核心原则

促单的关键是：**制造紧迫感 + 塑造价值 + 明确行动指令**

## 促单话术类型

### 1. 限时限量型
```
"今天直播间专属价，只有在我这里才能买到这个价格！"
"库存只剩最后50单，抢完恢复原价！"
"这个价格只维持到今天直播结束，明天来都没有！"
```

### 2. 价值对比型
```
"同款产品其他直播间卖199，今天我给你们99，还送XX赠品！"
"专柜价格699，今天直播间399，相当于5折！"
"成本价给你们，就为了涨粉，不赚钱！"
```

### 3. 身份认同型
```
"今天能进来的都是我的老粉，老粉都知道我的风格！"
"真正识货的宝宝才会在这个时间进来！"
"能刷到我的都是有缘人，今天不让你们空手回去！"
```

### 4. 行动指令型
```
"想要的把'要'打在公屏上！"
"没关注的点下关注，我改价！"
"把定金打在评论区，我给你们上链接！"
```

## 促单节奏把控

### 开场15分钟
- 建立信任，介绍自己
- 不要急着卖货，先暖场

### 30-60分钟
- 正式进入产品介绍
- FAB法则使用
- 穿插互动和福利

### 最后的促单环节
- 强调限时限量
- 快速重复核心卖点
- 倒计时制造紧迫感

## 话术模板

### 标准促单话术
```
来！家人们！重点来了！
今天这款产品我给你们 XXX钱！
外面你买都买不到这个价格！
我今天不赚钱，就为了交个朋友！
想要的家人们把"想要"打在评论区！
不想要的也没关系，看一看嘛！
```

### 逼单话术
```
好！倒计时开始！
54321！
手慢的等下一批！
没抢到的家人们别急，我再给你们补20单！
```

## 注意事项

1. 不要过度催促，会让用户反感
2. 价值塑造要真实，不要虚假宣传
3. 限时限量要说具体数字，不要模糊
4. 促单时情绪要饱满，但不要夸张
5. 注意语速，不要太快，让用户听清
