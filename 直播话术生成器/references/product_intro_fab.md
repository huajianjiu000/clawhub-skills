# FAB法则产品介绍指南

## FAB法则定义

- **F**eature（特性）：产品的客观属性
- **A**dvantage（优势）：这些属性带来的优点
- **B**enefit（利益）：对用户有什么好处

## FAB转换技巧

### 1. 从特性到优势
| 特性 | 优势 |
|------|------|
| 纯棉材质 | 透气性好，穿着舒适 |
| 大容量电池 | 续航时间长，不用频繁充电 |
| 智能芯片 | 运行速度快，不卡顿 |

### 2. 从优势到利益
| 优势 | 利益（对用户的好处）|
|------|------|
| 透气性好 | 夏天不闷热，运动更舒服 |
| 续航长 | 出差旅行不用带充电宝 |
| 运行快 | 工作效率高，心情更舒畅 |

## FAB话术公式

```
【特性】我们这款XX，采用的是XX（材质/技术/工艺）...
【优势】这个配置让XX（产品名）比同类产品更XX...
【利益】所以你用的时候，可以XX（具体好处）...
```

## 直播话术示例

### 护肤品
```
【特性】这支精华液含有10%的烟酰胺成分
【优势】这个浓度是市面上最高的，美白效果是普通产品的3倍
【利益】用完一瓶就能看到明显的肤色提亮，坚持使用让你白到发光！
```

### 食品
```
【特性】这款坚果来自云南深山，海拔2000米以上
【优势】昼夜温差大让坚果积累更多营养，口感更香脆
【利益】每一颗都是大自然的馈赠，吃出健康好气色！
```

### 电子产品
```
【特性】这款手机配备了5000毫安的超大电池
【优势】可以连续玩12小时游戏，看24小时视频
【利益】出门再也不用带充电宝了，告别电量焦虑！
```

## 注意事项

1. 先说利益，再说特性和优势（利益前置）
2. 用具体的数字和对比增加可信度
3. 结合用户使用场景描述利益
4. 语气要热情，有感染力
