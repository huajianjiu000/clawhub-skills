# 直播话术生成器 (Live Stream Script Generator)

## 简介

专为抖音、快手、视频号带货主播设计的直播话术生成工具，帮助主播快速生成专业的直播话术，提升带货转化率。

## 适用场景

- 带货直播开场
- 产品介绍话术
- 促单逼单话术
- 互动留人话术
- 收尾感谢话术

## 触发词

```
直播话术, 带货脚本, 直播脚本, 主播话术, 直播开场白, 产品介绍, 促单话术, 催单话术, 留人话术, 互动话术, 直播收尾
```

## 使用方法

用户输入产品信息、直播场景、目标受众后，生成对应的话术脚本。

## 输出格式

### 标准直播话术脚本

**基本信息**
- 产品名称：[用户输入]
- 适用平台：[抖音/快手/视频号]
- 话术类型：[开场/产品/促单/留人/收尾]
- 建议时长：[X分钟]

**完整话术**
[生成的专业话术内容]

**注意事项**
- 语速建议：[X字/秒]
- 情绪要点：[热情/专业/紧迫感]
- 互动提示：[提问/引导评论]

## 核心功能

### 1. 开场话术
- 热场互动
- 福利预告
- 建立信任

### 2. 产品介绍话术
- FAB法则（特性-优势-利益）
- 场景化描述
- 对比展示

### 3. 促单逼单话术
- 限时限量
- 价值塑造
- 行动指令

### 4. 互动留人话术
- 提问互动
- 福利留人
- 节奏把控

### 5. 收尾话术
- 感谢互动
- 下播预告
- 粉丝维护
