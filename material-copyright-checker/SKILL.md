## 功能说明

### 核心功能
1. **版权类型判断** - 判断素材是商用免费还是需要授权
2. **平台规则查询** - 各平台对素材的使用规定
3. **替代方案推荐** - 提供免费可商用的替代素材
4. **风险提示** - 提醒可能存在的侵权风险

### 支持查询的素材类型
- 🖼️ 图片（封面、配图、海报）
- 🎵 音乐（BGM、配乐）
- 🎬 视频片段（混剪素材）
- 📝 文案（台词、字幕）

## 版权类型说明

### CC0协议（完全免费）
- 可自由使用，无需署名
- 可商用，无需付费
- 代表平台：Unsplash、Pexels、Pixabay

### CC BY协议（署名即可）
- 使用时需注明原作者
- 可商用，无需付费
- 代表平台：Flickr部分图片

### 付费授权
- 需要购买授权
- 不可免费使用
- 代表平台：视觉中国、东方IC

### 平台限制
- 部分素材仅限平台内使用
- 不可跨平台搬运

## 快速判断指南

### 图片版权快速判断
| 来源 | 是否可商用 |
|------|-----------|
| 自己拍摄/制作 | ✅ 可以 |
| 免费图库（CC0） | ✅ 可以 |
| 小红书/微博/抖音博主 | ❌ 不可以 |
| 百度图片 | ❌ 不可以 |
| 视觉中国 | ❌ 需付费 |
| Google图片 | ❌ 不可以 |

### 音乐版权快速判断
| 来源 | 是否可商用 |
|------|-----------|
| 平台自带音乐库 | ✅ 可以（限本平台） |
| CC0音乐 | ✅ 可以 |
| QQ音乐/网易云付费歌曲 | ❌ 不可以 |
| 热门影视原声 | ❌ 不可以 |
| 自行创作/购买授权 | ✅ 可以 |
