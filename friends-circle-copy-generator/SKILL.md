# 朋友圈爆款文案生成器 (Viral Moments Copy Generator)

## 技能概述
帮助用户生成高互动、高点赞的朋友圈文案，适用于日常分享、产品推广、活动宣传等多种场景。

## 核心功能

### 1. 场景化文案生成
针对不同朋友圈场景生成文案：
- 日常分享：生活感悟、旅行打卡、美食分享
- 产品推广：微商文案、种草笔记、限时福利
- 活动宣传：邀请函、抽奖活动、团购招募
- 人设打造：职场精英、宝妈分享、创业故事
- 情感表达：生日祝福、节日问候、思念文案

### 2. 爆款元素融入
自动融入朋友圈爆款元素：
- 悬念式开头（引发好奇）
- 数字量化（"3个技巧"）
- 情绪共鸣（引发评论）
- 互动引导（@某人、投票）
- 限时紧迫感（活动文案）

### 3. 配套素材建议
除了文案，还提供：
- emoji使用建议
- 图片数量建议
- 发布时间建议
- 互动话术回复模板

## 适用场景
- 微商/电商朋友圈运营
- 个人IP人设打造
- 社群引流转化
- 日常生活记录

## 使用方法
用户输入：场景类型 + 核心信息 → 输出文案+配套建议

## 输出格式
```
📱 朋友圈文案推荐

[主文案]

配套建议：
🕐 发布时间：[建议时间]
🖼️ 图片数量：[建议数量]
😊 emoji使用：[建议]
💬 互动引导：[引导语]

📋 变体版本
版本1：[文案风格1]
版本2：[文案风格2]
版本3：[文案风格3]

💡 发布小技巧
1. [技巧1]
2. [技巧2]
```
