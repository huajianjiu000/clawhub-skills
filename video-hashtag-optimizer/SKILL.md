# Video Hashtag Optimizer (视频标签优化器)

## 1. Concept & Vision

**视频标签优化器** - 帮助创作者优化视频标签，获取更多推荐流量的智能工具。标签选对，流量翻倍；标签用好，自然流量少不了。

**产品口号**: "选对标签，让算法帮你推流"

## 2. Core Features

### 2.1 标签分析功能
- **标签热度分析**: 实时查询标签的搜索量和竞争度
- **标签组合推荐**: 根据内容推荐最优标签组合
- **竞品标签分析**: 分析同类爆款视频的标签策略
- **标签趋势追踪**: 追踪近期热门上升标签

### 2.2 标签分类体系
- **主流量标签**: 大流量池标签（1000万+播放）
- **垂直精准标签**: 精准人群标签（100-500万播放）
- **长尾潜力标签**: 上升期标签，流量增长快
- **热点借势标签**: 蹭热点专用，快速获取流量

### 2.3 平台差异化策略
- **抖音**: 注重算法推荐机制
- **快手**: 注重私域流量运营
- **视频号**: 注重社交推荐机制
- **B站**: 注重内容垂直深度

### 2.4 输出内容
```
【标签组合】推荐使用的完整标签列表
【标签分类】每个标签的流量级别
【使用建议】标签排序和数量建议
【注意事项】平台禁忌标签提醒
```

## 3. User Interaction Flow

```
输入视频主题 → 选择发布平台 → 生成标签方案 → 根据建议调整使用
```

**用户输入示例**:
- "职场晋升干货" + "抖音" → 输出一套完整的抖音标签组合

## 4. Examples

**输入**: 美食教程 - 家常川菜
**输出**:
```markdown
【推荐标签组合】
#川菜 #家常菜 #下饭菜 #美食教程 #厨房小白
#简单家常菜 #川菜做法 #今天吃什么 #美食推荐
#厨房技巧 #新手做饭 #家庭美食

【标签分析】
🔥 高热度标签: #川菜(5000万+) #美食教程(2000万+)
🎯 精准标签: #家常川菜 #川菜做法 #厨房小白
📈 潜力标签: #家庭美食 #下饭菜做法
⚠️ 借势标签: (当前无热点可蹭)

【使用建议】
1. 主标签放前3个（#川菜 #家常菜 #下饭菜）
2. 标签数量: 抖音建议8-12个
3. 避免标签堆砌，同类型选最热的一个即可

【平台适配】
- 抖音: 侧重算法推荐，多用热门标签
- 快手: 多加 #老铁 #家人 等社区标签
- 视频号: 多用 #美食 #教程 等通用标签
```

## 5. Technical Specifications

- **输入**: 视频内容主题 + 发布平台
- **处理**: 基于平台标签数据库 + 热度算法
- **输出**: 最优标签组合 + 使用策略
- **语言**: 中文
