## 功能说明

### 核心功能
1. **平台分析** - 分析各平台用户活跃时间
2. **内容匹配** - 根据内容类型推荐最佳时段
3. **时段对比** - 对比工作日vs周末差异
4. **实时推荐** - 结合当前时间给出即时建议

## 使用场景

### 场景1：日常内容发布
```
用户：我是做美食探店的，应该什么时候发抖音？
助手：🍜 美食探店类内容推荐：

📱 抖音最佳发布时间：
• 工作日：17:00-20:00（下班觅食高峰期）
• 周末：11:00-13:00（午餐探店）, 18:00-21:00（晚餐探店）

💡 建议：美食类内容在饭点前1-2小时发布效果最佳
⏰ 当前时间推荐：如果是今天，可以立即发布
```

### 场景2：直播带货
```
用户：准备在快手直播卖女装，什么时间最好？
助手：👗 快手女装直播推荐：

📺 推荐时段：
• 晚间场：20:00-23:00（黄金时段，转化率最高）
• 下午场：14:00-17:00（适合日常款）
• 周末加开：12:00-14:00

💡 建议：提前30分钟发预告短视频预热
```
