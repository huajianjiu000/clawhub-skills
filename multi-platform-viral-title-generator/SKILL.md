# 多平台爆款标题生成器

## 功能描述

专为短视频创作者打造的多平台爆款标题生成工具，支持抖音、快手、视频号、B站、头条五大平台，自动适配各平台标题风格和算法偏好。

## 适用场景

- 短视频创作者需要为不同平台创作爆款标题
- 内容运营人员需要批量生成多平台适配标题
- 自媒体人需要提升内容点击率和曝光量

## 核心能力

### 支持平台
- **抖音**：15-30字，突出悬念和情绪， hashtag 优化
- **快手**：接地气、烟火气，突出真实感和共鸣
- **视频号**：30字以内，强调价值感和情感连接
- **B站**：可较长，支持系列感标题
- **头条号**：新闻感、资讯感，突出信息价值

### 功能特点
1. **多平台一键生成**：输入主题或关键词，同时生成5个平台的适配标题
2. **平台风格识别**：自动识别各平台的标题规范和算法偏好
3. **爆款元素植入**：内置悬念、数字、情绪、痛点等爆款元素
4. **标签推荐**：为每个标题推荐合适的 hashtag 或话题标签
5. **批量生成**：支持同一主题生成多组标题供选择

## 使用方式

### 基本用法
```
请帮我生成[你的视频主题]的多平台标题
```

### 进阶用法
```
请生成关于[主题]的爆款标题，要求：
- 情绪基调：[正面/负面/中立]
- 目标受众：[人群描述]
- 内容类型：[知识分享/娱乐/情感/种草等]
```

## 输入参数

| 参数 | 必填 | 说明 |
|------|------|------|
| 视频主题 | 是 | 视频核心内容或关键词 |
| 目标平台 | 否 | 指定平台，默认为全平台 |
| 情绪基调 | 否 | 正面/负面/中立 |
| 数量 | 否 | 生成标题数量，默认5个 |

## 输出格式

```markdown

1. [标题] #话题1 #话题2
2. [标题] #话题1 #话题2

## 快手标题

2. [标题] $话题1 $话题2

## 视频号标题

## 头条号标题
1. [标题]

```

## 技术实现

本技能通过以下方式工作：

1. **平台特征分析**：分析各平台用户画像和内容偏好
2. **标题结构拆解**：将爆款标题拆解为「悬念+数字+情绪+价值」等元素
3. **智能重组生成**：根据用户输入重组元素，生成平台适配标题
4. **标签优化**：根据平台算法推荐最优标签组合
