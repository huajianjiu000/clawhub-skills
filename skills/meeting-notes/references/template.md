# 会议纪要模板

## 标准化会议纪要格式

### 基本信息模块
- 会议名称/主题
- 会议时间（开始-结束）
- 会议地点
- 主持人
- 记录人
- 参与者（列出姓名和角色）

### 议程模块
1. 上一阶段工作回顾
2. 本次会议议题
3. 讨论内容摘要
4. 决议事项
5. 下一步工作安排

### 决策记录模块
- 决策编号
- 决策内容
- 决策依据
- 责任人
- 完成标准

### 行动项模块
| # | 任务描述 | 负责人 | 协助人 | 开始时间 | 截止时间 | 优先级 | 状态 |
|---|----------|--------|--------|----------|----------|--------|------|
| 1 |          |        |        |          |          |        |      |

### 风险与问题模块
- 问题描述
- 影响分析
- 应对策略
- 升级条件

### 会议评估模块
- 会议效率评分（1-5）
- 议程覆盖率
- 决策完成率
- 改进建议
