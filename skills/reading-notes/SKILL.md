# 阅读笔记整理 (Reading Notes Organizer)

## 技能描述
智能整理阅读内容，提取核心知识点，生成结构化笔记。支持书籍、文章、论文等多种格式，自动提炼精华和行动指南。

## 核心功能
- 智能提取核心概念
- 生成知识点思维导图
- 提炼金句和案例
- 创建读书笔记卡片
- 提供实践应用建议

## 适用场景
- 书籍精读笔记
- 论文核心提炼
- 文章要点整理
- 课程学习笔记
- 行业报告解读

## 技能结构
```
reading-notes/
├── SKILL.md              # 主文件
├── references/
│   └── note_templates.md # 笔记模板
└── scripts/
    └── organize_notes.py # 整理脚本
```

## 笔记整理结构
```markdown
# [书名/文章名]读书笔记

## 基本信息
## 核心主题
## 主要观点
## 精华摘录
## 案例整理
## 思维导图
## 实践应用
## 个人感悟
```

## 使用方法
1. 提供阅读材料内容或书名
2. 说明阅读目的（学习/消遣/研究）
3. 指定笔记深度（概要/详细/精华）
4. 可选：标注重点关注章节
5. 系统生成完整读书笔记
