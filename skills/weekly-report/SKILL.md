# 周报自动生成器 (Weekly Report Generator)

## 技能描述
根据工作内容和项目进展自动生成结构化周报。支持多种格式输出，自动统计工作量，便于汇报和复盘。

## 核心功能
- 智能识别工作内容分类
- 自动统计工作量和完成情况
- 生成清晰的进度可视化
- 支持多种周报格式
- 自动生成下周计划

## 适用场景
- 互联网/科技公司周报
- 项目管理周报
- 销售/运营周报
- 个人工作总结
- 团队/部门周报

## 周报标准格式
```markdown
# 周报 [姓名] - [日期]

## 本周工作总结
## 重点项目进展
## 数据统计
## 问题与风险
## 下周工作计划
```

## 使用方法
1. 简要描述本周完成的工作
2. 提供项目进展（如有）
3. 指定周报格式偏好
4. 可选：添加数据指标
5. 系统自动生成完整周报

## 支持的周报类型
| 类型 | 适用人群 | 特点 |
|------|----------|------|
| 研发周报 | 开发/测试 | 含代码/测试数据 |
| 产品周报 | 产品经理 | 含需求/上线数据 |
| 运营周报 | 运营人员 | 含活动/用户数据 |
| 销售周报 | 销售人员 | 含业绩/客户数据 |
| 通用周报 | 通用场景 | 标准化格式 |
