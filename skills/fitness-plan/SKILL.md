# 健身计划定制 (Fitness Plan Generator)

## 技能描述
根据身体状况和健身目标，智能生成科学、个性化的健身计划。支持增肌、减脂、塑形等多种目标，自动包含训练方案和饮食建议。

## 核心功能
- 评估身体状况和健身基础
- 制定个性化训练计划
- 提供饮食和营养建议
- 包含热身和拉伸指导
- 跟踪进度和调整建议

## 适用场景
- 增肌训练计划
- 减脂瘦身计划
- 塑形美体计划
- 体能提升计划
- 康复恢复计划

## 健身计划结构
```markdown
# [目标]健身计划

## 健身目标
## 身体评估
## 训练计划
## 饮食建议
## 注意事项
## 进度跟踪
```

## 使用方法
1. 描述健身目标（减脂/增肌/塑形）
2. 提供基本信息（身高/体重/年龄/性别）
3. 说明健身基础（新手/有经验）
4. 指定可用训练设备
5. 系统生成个性化健身计划

## 健身目标类型
| 目标 | 特点 | 训练重点 |
|------|------|----------|
| 减脂 | 降低体脂率 | 有氧+力量 |
| 增肌 | 增加肌肉量 | 力量训练为主 |
| 塑形 | 改善体型 | 力量+拉伸 |
| 体能 | 提升耐力 | 综合训练 |
