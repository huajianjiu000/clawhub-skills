# 产品PRD生成器 (Product PRD Generator)

## 技能描述
快速生成专业、完整的产品需求文档（PRD）。支持从概念到详细设计的全流程文档生成，自动包含用户故事、功能规格和非功能性需求。

## 核心功能
- 自动生成PRD标准结构
- 智能识别产品定位和目标用户
- 生成完整的用户故事和用例
- 定义清晰的验收标准
- 支持多产品类型（App、Web、SaaS等）

## 适用场景
- 创业项目PRD撰写
- 产品迭代需求文档
- 功能优化方案设计
- MVP产品规划
- 投标方案文档

## 技能结构
```
prd-generator/
├── SKILL.md              # 主文件
├── references/
│   └── prd_template.md   # PRD模板
└── scripts/
    └── generate_prd.py   # 生成脚本
```

## PRD标准结构
```markdown
# 产品需求文档 (PRD)

## 1. 产品概述
## 2. 目标与范围
## 3. 用户分析
## 4. 功能需求
## 5. 非功能需求
## 6. UI/UX要求
## 7. 验收标准
## 8. 风险评估
## 9. 附录
```

## 使用方法
1. 描述产品基本概念和核心价值
2. 提供目标用户画像（可选）
3. 说明主要功能需求
4. 指定产品类型和平台
5. 系统自动生成完整PRD
