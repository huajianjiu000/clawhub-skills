# 邮件智能回复 (Email Smart Reply)

## 技能描述
根据邮件内容智能生成专业、得体的回复。支持多种场景（商务、客服、求职、感谢等），自动匹配语气和风格。

## 核心功能
- 智能分析原邮件意图和情感倾向
- 生成多种语气风格的回复选项
- 自动补全称呼和署名
- 支持多语言回复
- 提供修改建议和优化

## 适用场景
- 商务邮件往来
- 客户服务回复
- 求职申请回复
- 会议邀请回复
- 投诉/感谢邮件处理


## 使用方法
1. 粘贴原始邮件内容
2. 选择回复场景（商务/客服/求职/感谢等）
3. 指定语气偏好（正式/友好/简洁/详细）
4. 可选：提供关键信息（姓名、公司等）
5. 系统生成专业回复

## 输出示例
```
主题：Re: 产品报价咨询

尊敬的[姓名]：

您好！

感谢您的来信询问。

[回复内容]

期待您的进一步反馈。

此致
敬礼

[您的姓名]
[日期]
```
