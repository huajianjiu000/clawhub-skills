# 邮件回复模板库

## 商务邮件模板

### 初次联系
```
尊敬的[姓名]：

您好！

很荣幸通过[来源]了解到贵司[业务]。我们是[公司名称]，专注于[领域]。

[自我介绍和价值主张]

希望能有机会与您进一步交流，探讨合作可能。

期待您的回复！

此致
敬礼

[您的姓名]
[职位]
[联系方式]
```

### 报价回复
```
尊敬的[姓名]：

您好！

感谢您对我司[产品/服务]的关注。

针对您的需求，我们提供以下方案：

[方案详情]

- [优势1]
- [优势2]
- [优势3]

价格说明：[价格细节]
有效期：[有效期限]

如有任何疑问，请随时联系我。

祝商祺！

[您的姓名]
[公司名称]
```

### 拒绝回复
```
尊敬的[姓名]：

您好！

感谢您的来信和宝贵建议。

[委婉说明原因]

虽然目前无法[满足需求]，但我们非常重视您的意见。

[提供替代方案或建议]

再次感谢您的理解与支持！

此致
敬礼

[您的姓名]
```

## 高频短语库

### 开场白
- 感谢您的来信/咨询/关注
- 很高兴收到您的回复
- 非常感谢您的耐心等待
- 收到您的邮件，回复如下

### 正文过渡
- 针对您的问题
- 关于您提到的
- 根据您的需求
- 我们的建议是

### 结尾语
- 期待您的回复
- 如有疑问请随时联系
- 感谢您的理解与支持
- 祝您生活愉快/工作顺利

### 结束语
- 此致敬礼（正式）
- 祝好（一般）
- 顺祝商祺（商务）
- 致以诚挚的问候
