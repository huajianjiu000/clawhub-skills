# 旅行攻略生成器 (Travel Guide Generator)

## 技能描述
根据目的地和个人偏好，智能生成完整的旅行攻略。包含行程规划、景点推荐、美食指南、交通建议和预算预估。

## 核心功能
- 智能规划行程路线
- 推荐必去景点和小众目的地
- 提供美食和购物指南
- 交通方式和时间规划
- 预算费用预估

## 适用场景
- 国内自由行规划
- 出境游准备
- 周末短途游
- 亲子/家庭游
- 蜜月/情侣游

## 旅行攻略结构
```markdown
# [目的地]旅行攻略

## 基本信息
## 行程概览
## 每日行程
## 景点详解
## 美食推荐
## 交通指南
## 住宿建议
## 预算清单
## 注意事项
```

## 使用方法
1. 提供目的地名称
2. 说明旅行天数和人数
3. 指定旅行偏好（穷游/品质游/亲子等）
4. 标注特殊需求（必去景点/美食要求）
5. 系统生成完整旅行攻略

## 旅行类型
| 类型 | 特点 | 预算范围 |
|------|------|----------|
| 穷游 | 经济实惠 | 300-500/天/人 |
| 品质游 | 舒适体验 | 800-1500/天/人 |
| 豪华游 | 高端享受 | 2000+/天/人 |
| 亲子游 | 适合儿童 | 需亲子设施 |
| 蜜月游 | 浪漫氛围 | 品质优先 |
