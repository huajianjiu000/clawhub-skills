# 合同风险审查清单

## 一、通用风险点清单

### 1.1 主体资格风险
- [ ] 对方是否为合法注册的公司
- [ ] 对方签约人是否有授权
- [ ] 合同主体名称是否与公章一致
- [ ] 对方是否存在经营异常

### 1.2 标的物/服务风险
- [ ] 标的物/服务描述是否清晰
- [ ] 规格、数量是否明确
- [ ] 质量标准是否有明确约定
- [ ] 交付标准是否可量化

### 1.3 价款支付风险
- [ ] 价款金额是否明确（含税/不含税）
- [ ] 支付方式（一次性/分期/里程碑）
- [ ] 支付时间节点是否合理
- [ ] 是否有履约保证金条款
- [ ] 付款条件是否过于苛刻

### 1.4 履约期限风险
- [ ] 履行期限是否明确
- [ ] 是否有宽限期或延期条款
- [ ] 不可抗力条款是否合理
- [ ] 是否有验收期限

### 1.5 违约责任风险
- [ ] 违约金比例是否过高
- [ ] 是否有加重己方责任的条款
- [ ] 违约情形是否列举完整
- [ ] 是否有单方解除权条款
- [ ] 免责条款是否过于宽泛

### 1.6 知识产权风险
- [ ] 知识产权归属是否明确
- [ ] 是否有竞业限制条款
- [ ] 是否有保密条款
- [ ] 授权范围是否清晰

### 1.7 争议解决风险
- [ ] 管辖法院/仲裁机构是否对我方有利
- [ ] 是否约定过高维权成本
- [ ] 是否有人数众多诉讼条款

## 二、劳动合同特别审查点

### 2.1 必备条款
- [ ] 劳动合同期限
- [ ] 工作内容和工作地点
- [ ] 工作时间和休息休假
- [ ] 劳动报酬
- [ ] 社会保险
- [ ] 劳动保护和劳动条件

### 2.2 常见风险条款
| 风险类型 | 风险描述 | 应对建议 |
|----------|----------|----------|
| 试用期过长 | 超过6个月 | 缩短试用期 |
| 工资低于约定 | 实际发放与约定不符 | 明确工资结构 |
| 加班无限制 | 约定无条件加班 | 限制加班时间 |
| 竞业限制过宽 | 限制过严、补偿过低 | 缩小范围、提高补偿 |
| 违约金条款 | 设置不合理违约金 | 删除或修改 |

### 2.3 违法条款识别
- ❌ 不得约定"工伤自负"
- ❌ 不得约定"女员工N年内不得怀孕"
- ❌ 不得约定"放弃社保"
- ❌ 不得约定"无条件服从加班"

## 三、商务合同特别审查点

### 3.1 采购合同
| 条款 | 关注点 | 风险等级 |
|------|--------|----------|
| 质量标准 | 是否明确、可检测 | ⭐⭐⭐ |
| 验收标准 | 验收流程、异议期 | ⭐⭐⭐ |
| 交货期 | 延误责任划分 | ⭐⭐⭐ |
| 付款条件 | 先票后款风险 | ⭐⭐ |
| 质量保证金 | 比例、期限 | ⭐⭐ |

### 3.2 销售合同
| 条款 | 关注点 | 风险等级 |
|------|--------|----------|
| 所有权保留 | 是否约定 | ⭐⭐⭐ |
| 赊销额度 | 信用管控 | ⭐⭐ |
| 退货条款 | 条件、时限 | ⭐⭐⭐ |
| 窜货管控 | 区域限制 | ⭐⭐ |

### 3.3 租赁合同
| 条款 | 关注点 | 风险等级 |
|------|--------|----------|
| 押金条款 | 退还条件、时限 | ⭐⭐⭐ |
| 提前解约 | 违约金是否过高 | ⭐⭐⭐ |
| 维修责任 | 划分是否合理 | ⭐⭐ |
| 租金递增 | 涨幅是否受限 | ⭐⭐ |

### 3.4 服务合同
| 条款 | 关注点 | 风险等级 |
|------|--------|----------|
| 服务标准 | 是否可量化 | ⭐⭐⭐ |
| 知识产权 | 成果归属 | ⭐⭐⭐ |
| 保密义务 | 范围和期限 | ⭐⭐ |
| 竞业限制 | 合理性审查 | ⭐⭐⭐ |

## 四、风险等级说明

| 等级 | 标识 | 说明 | 建议 |
|------|------|------|------|
| 高风险 | 🔴 | 可能有重大损失 | 必须修改 |
| 中风险 | 🟡 | 可能造成一定损失 | 建议修改 |
| 低风险 | 🟢 | 风险较小 | 可接受或协商 |

## 六、常用法律依据

### 《中华人民共和国民法典》
- 合同编通则：第463-509条
- 买卖合同：第595-638条
- 租赁合同：第703-734条
- 委托合同：第919-936条

### 《中华人民共和国劳动合同法》
- 合同订立：第10-28条
- 合同期限：第12-16条
- 试用期：第19-21条
- 违约金：第22-25条
- 解除合同：第36-44条

### 《中华人民共和国消费者权益保护法》
- 知情权：第8条
- 自主选择权：第9条
- 公平交易权：第10条
