# 合同条款审查 (Contract Clause Reviewer)

## 技能描述
智能审查合同条款，识别潜在风险点，提供修改建议。支持多种合同类型，帮助非法律专业人士快速识别关键问题。

## 核心功能
- 识别合同关键条款
- 标注潜在风险点
- 提供法律依据建议
- 生成修改建议文本
- 提示需要注意的陷阱

## 适用场景
- 劳动合同审查
- 商务合同审核
- 租赁合同检查
- 采购合同分析
- 合作协议评估

## 审查维度
```markdown
## 1. 合同主体审查
## 2. 标的与质量条款
## 3. 权利义务条款
## 4. 违约责任条款
## 5. 争议解决条款
## 6. 特殊约定审查
```

## 使用方法
1. 提供合同文本内容
2. 指定合同类型
3. 说明己方立场（甲方/乙方）
4. 可选：提供对方背景信息
5. 系统生成审查报告

## 支持的合同类型
| 类型 | 特点 | 审查重点 |
|------|------|----------|
| 劳动合同 | 保护劳动者 | 薪酬、工时、解约 |
| 商务合同 | 对等协商 | 权责、违约、争议 |
| 租赁合同 | 保护出租方 | 押金、维修、提前解约 |
| 采购合同 | 保护采购方 | 质量、交期、验收 |
| 投资合同 | 高风险 | 股权、对赌、退出 |
