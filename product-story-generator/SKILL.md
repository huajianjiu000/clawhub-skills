# 产品故事生成器 (Product Story Generator)

## 产品定位
帮助品牌方、营销人员快速生成有感染力的产品故事。通过讲述产品背后的故事、创始人初心、用户案例等内容，建立品牌情感连接，提升产品附加值和用户认同感。

## 核心功能
- 品牌故事生成
- 创始人故事生成
- 产品诞生故事
- 用户案例故事
- 品牌slogan生成
- 品牌理念文案

## 故事类型

### 1. 品牌故事
- 品牌创立背景
- 创始人初心与愿景
- 品牌使命与价值观
- 发展里程碑事件

### 2. 产品故事
- 产品灵感来源
- 研发历程（困难与突破）
- 产品与用户的故事
- 产品解决的痛点

### 3. 创始人故事
- 个人背景与经历
- 创业动机
- 关键转折点
- 价值观与信念

### 4. 用户案例
- 用户画像描述
- 使用场景还原
- 改变与收获
- 真实反馈引用

## 输出格式

### 品牌故事格式
```
# [品牌名称] - 让[愿景/使命]

## 初心
[为什么创立这个品牌]

## 坚持
[品牌的核心价值观/坚持]

## 承诺
[对用户的承诺/保证]

[可选：创始人寄语]
```

### 产品故事格式
```
# [产品名称] - [产品定位/价值主张]

## 灵感
[产品诞生的故事/灵感来源]

## 打磨
[研发/生产过程中的故事]

## 用户
[典型用户与产品的故事]

## 意义
[产品背后的意义/价值]
```

## 故事结构公式

### AACT 公式
- **A**ction（行动）：谁做了什么
- **A**ctually（实际上）：背后的原因
- **C**onnect（连接）：与用户的共鸣
- **T**ransform（转变）：带来的改变

### 三幕式结构
1. **第一幕：冲突** - 面临的问题/挑战
2. **第二幕：挣扎** - 寻找解决方案的过程
3. **第三幕：突破** - 最终的成功/改变
